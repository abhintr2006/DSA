# Security Policy

## Supported Versions

The following versions of DSA Study Hub are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please let us know immediately. We take security seriously and will investigate all reports.

Please report vulnerabilities by:

1. Opening a **GitHub Issue** with the label `security` (avoid including sensitive details in the public issue).
2. Or contact the maintainers directly via the contact information provided in the **About Us** section of the application.

Please include the following in your report:

- Type of issue (e.g., XSS, SQL injection, etc.)
- Location/URL where the vulnerability exists
- A brief description of the vulnerability
- Steps to reproduce (if possible)

We will acknowledge your report within 48 hours and provide a timeline for a fix. Please do not disclose the vulnerability publicly until a fix has been released.
